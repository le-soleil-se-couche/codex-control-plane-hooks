# Code of Conduct

## Our standard

Participate with respect, technical honesty, and care for other people's privacy. Constructive disagreement is welcome. Harassment, intimidation, discriminatory conduct, and publication of private information are unacceptable.

## Enforcement

Repository maintainers may edit or remove contributions and restrict participation when conduct threatens people, project integrity, or privacy. Report conduct concerns through a private repository contact channel when available.

This compact policy may be replaced by a formally adopted community code as the project grows.
