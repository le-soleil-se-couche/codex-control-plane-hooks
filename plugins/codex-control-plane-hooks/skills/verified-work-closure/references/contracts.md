# Delivery receipt contract

For material work, report:

## Artifacts

- Exact files, repository URL, commit, or other durable outputs.

## Checks

- Commands or inspections actually run.
- Result and scope of each check.

## Notes

- Residual risk, rollback path, and `[UNRUN]` items.

Never include credentials, private customer records, live holdings, account data, or other configured sensitive-business details in the receipt.
